# 行政書士事業計画シミュレーター

報酬額統計を参考に、行政書士事務所の年間売上、粗利益、営業利益、必要問い合わせ件数、稼働率、損益分岐点を試算するMVPです。

## セットアップ

```bash
npm install
npm run dev
```

テストと本番ビルド：

```bash
npm test
npm run build
```

## 実装済み

- 令和7年度報酬額統計の全487業務を、業務名・カテゴリー・統計番号で検索
- 回答者数、平均額、最頻値、最小値、最大値、詳細分布の表示
- 業務別の報酬、件数、原価、時間、成約率設定
- 統計にない任意業務の作成と計画・JSONへの保存
- 売上、粗利益、営業利益、時間単価、問い合わせ数の自動計算
- 固定費、売上比率・件数連動の変動経費、生活費、内部留保、納税準備額と損益分岐点
- 3方式の目標逆算
- 最大20件のLocalStorage保存、JSON入出力（1MB制限）、印刷
- レスポンシブUI

## データについて

日本行政書士会連合会「令和7年度報酬額統計調査の結果」の全487業務を収録しています。金額は税込、立替金を除く統計値です。回答者ゼロの項目、複数の最頻値、取扱者に制限のある項目も元資料に合わせて保持しています。

配信時は `src/services.generated.json` を静的データとして読み込み、PDFへの通信や実行時スクレイピングは行いません。統計更新時は公式PDFを用意し、次の抽出スクリプトで再生成できます。

```bash
python scripts/extract_reward_data.py path/to/reward-survey.pdf src/services.generated.json
```

スクリプトは、業務番号1～487の連続性、回答者数と金額帯回答数の一致、平均額と最小・最大額の整合性を検査します。

## Cloudflareへの展開

現在のMVPは静的なViteアプリで、`npm run build` の出力先 `dist` をCloudflare Pagesへデプロイできます。ログイン不要・単一ユーザー利用ではデータをブラウザ内に保存するためD1は不要です。公式統計マスタの一元配信やアカウント同期を追加する段階で、仕様書のD1スキーマとWorkers APIを導入してください。
